Blair One-Pager Editor

Build branded donor solicitation one-pagers without Canva or InDesign. Everything lives in one HTML file you open in your browser. No login, no uploads.

Building your page

Use the toolbar to add headings, subheads, body text and photo boxes. Drag any block to reposition it. Grab the orange corner handle to resize. Double-click text to edit it directly. The style dropdown in the right panel changes a block between heading, subhead and body. Press Ctrl+Z / Cmd+Z to undo and Ctrl+Y / Cmd+Y to redo.

Bulk paste text

Click Bulk paste text in the toolbar to populate all text blocks at once. Paste your content using labels on their own lines:

HEADING:  Main title
SUBHEAD:  Centered subtitle
BODY:  Opening paragraph (repeat for each body block)
SECTION:  Section subhead (repeat for each subhead)
QUOTE:  Donor quote text
ATTRIBUTION:  Quote attribution
BOXTITLE:  Callout box heading
BOXBODY:  Callout box body

Labels are case-insensitive and numbers are ignored, so BODY1 and BODY2 both work. Hit Apply to Page and your text drops into place. The pre-placed photo box automatically aligns with the first body paragraph and text flows around it.

Photos

Click "Upload photos" in the right panel to add images. Photo boxes fill top-to-bottom in the order you select files. Extras go to the thumbnail library. To fill a photo box from the library, select the box first then click a thumbnail. Select a photo block to reveal Landscape / Portrait orientation buttons and zoom controls. Drag inside the box to reposition. Zoom resets to 100% when you switch orientation.

Card strip

Use Change in the toolbar and choose Card strip to add a row of giving-level cards. This removes the quote bar. The strip auto-sizes to fill the remaining page. Click anywhere on the strip to select it — a Card photo panel appears in the right sidebar with two options: upload a new photo directly or pick from the library. Click a specific card to target that slot. Double-click any label or price to edit. Use + Card and − Card to add or remove cards.

Quote bar & callout box

Use Change → Quote bar or Change → Callout box to add a bottom element. If a card strip is present the quote bar is removed — use Change to bring it back above the strip. Double-click any text to edit it on the page.

Saving & exporting

Use Save to download a .blairproj file you can share or reopen later with Open. Use Download PDF to export a print-ready 8.5×11 PDF. Note: photo library images are stored in the browser session only — they are not saved in the .blairproj file, so re-upload photos after reopening a project.

Reset template

Click Reset template to start fresh. This clears all blocks, all text and the entire photo library.
